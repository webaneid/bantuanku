## üé® Color Harmony Groups

### Monochromatic Green
Untuk background yang halus dan tidak ramai:
- Ultra Light: `#E5F5EA` (`$primary-50`)
- Light Tint: `#4EBC7A` (`$primary-400`)
- Base: `#009B4C` (`$primary-500`)
- Dark: `#006833` (`$primary-700`)

### Analogous Harmony
Warna bersebelahan dalam roda warna untuk kesan tenang:
- Blue Green (Teal): `#008080`
- Darunnajah Green: `#009B4C`
- Lime Green: `#8CC63F`

### Complementary Colors
Untuk kontras tinggi (gunakan sparingly):
- Green: `#009B4C` (Primary)
- Red: `#C92424` (Danger - untuk notifikasi penting)
- Gold: `#F58220` (Accent - pasangan alami)

## üéØ Usage Guidelines

### When to Use Primary (Darunnajah Green)
```tsx
// Primary buttons
<button className="bg-primary-500 hover:bg-primary-600 text-white">
  Donasi Sekarang
</button>

// Active navigation
<a className="bg-primary-50 text-primary-600">Dashboard</a>

// Links
<a className="text-primary-500 hover:text-primary-600">Read More</a>

// Progress bars
<div className="bg-primary-500 h-2 rounded"></div>
```

### When to Use Secondary (Trust Blue)
```tsx
// Professional headings
<h1 className="text-secondary-700">Laporan Keuangan</h1>

// Secondary buttons (outline)
<button className="border-2 border-secondary-500 text-secondary-500 hover:bg-secondary-50">
  Lihat Detail
</button>

// Info badges
<span className="bg-secondary-100 text-secondary-700 px-2 py-1 rounded">
  Verified
</span>
```

### When to Use Accent (Laziswaf Gold)
```tsx
// CTA buttons with gradient
<button className="bg-gradient-gold text-white">
  Donasi Sekarang
</button>

// Urgent badges
<span className="bg-accent-500 text-white px-3 py-1 rounded-full">
  Urgent
</span>

// Featured content
<div className="border-l-4 border-accent-500 bg-accent-50 p-4">
  Featured Campaign
```

### Gradients

## üé® Color Combinations

### Combination 1: Primary + White
```css
background: var(--color-primary);
color: var(--color-white);
```
**Usage:** Primary buttons, headers, featured sections

### Combination 2: Secondary + White
```css
background: var(--color-secondary);
color: var(--color-white);
```
**Usage:** Secondary buttons, navigation, footer

### Combination 3: Light + Dark Text
```css
background: var(--color-light);
color: var(--color-dark);
```
**Usage:** Page backgrounds, cards

### Combination 4: White + Primary
```css
background: var(--color-white);
color: var(--color-primary);
border: 2px solid var(--color-primary);
```
**Usage:** Outline buttons, cards with borders

### Combination 5: Gradient (Primary ‚Üí Secondary)
```css
background: linear-gradient(135deg, 
  var(--color-primary) 0%, 
  var(--color-secondary) 100%
);
color: var(--color-white);
```
**Usage:** Hero sections, special banners

## üì± Accessibility

### Contrast Ratios (WCAG AA Compliant)

| Foreground | Background | Ratio | Pass |
|------------|-----------|-------|------|
| `#C9A961` (Primary) | `#FFFFFF` (White) | 3.5:1 | ‚úÖ Large text |
| `#006B5C` (Secondary) | `#FFFFFF` (White) | 7.2:1 | ‚úÖ All text |
| `#1A1A1A` (Dark) | `#FFFFFF` (White) | 16.1:1 | ‚úÖ All text |
| `#333333` (Text) | `#FFFFFF` (White) | 12.6:1 | ‚úÖ All text |
| `#FFFFFF` (White) | `#C9A961` (Primary) | 3.5:1 | ‚úÖ Large text |
| `#FFFFFF` (White) | `#006B5C` (Secondary) | 7.2:1 | ‚úÖ All text |

**Recommendations:**
- Use `--color-secondary` for regular text on white backgrounds (best contrast)
- Use `--color-primary` only for large text (18px+) or bold text on white
- Always use `--color-dark` or `--color-text` for body text
- Test with tools: [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)
